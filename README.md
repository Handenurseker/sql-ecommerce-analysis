# SQL E-Commerce Analysis

This project contains analytical SQL queries built on an e-commerce dataset.  
The goal of the project is to practice real-world business analysis scenarios using SQL.

## Dataset

The dataset represents a simplified e-commerce system including customer orders, addresses, cities, and payments.

Main tables used in this project:

- USERS
- ORDERS
- ADDRESS
- CITIES

## SQL Skills Demonstrated

This project demonstrates the use of:

- JOIN
- GROUP BY
- HAVING
- Aggregate functions (SUM, AVG, COUNT)
- Subqueries
- Derived tables
- Business analysis queries

## Purpose

The purpose of this repository is to demonstrate practical SQL analysis skills commonly used by Data Analysts when working with transactional datasets.
